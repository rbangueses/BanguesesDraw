# Dependency License Notices

BanguesesDraw embeds and depends on open-source packages, including the
open-source Excalidraw editor package `@excalidraw/excalidraw`.

Before distributing the app, generate and review dependency license notices
from the installed package tree and include required copyright and license text
for runtime dependencies.

The MVP is a local development app and keeps this file as the distribution
checklist for license compliance.
