Mermaid Flowchart Samples for a First Converter

These examples show the kind of compact Mermaid output we can ask AI to generate, then convert into editable Excalidraw boxes, labels, and arrows. The first pass would focus on simple flowcharts only.

Supported Diagram Patterns

A simple left-to-right workflow. Very safe for AI and easy to convert.

  D --> E[Closed won]

First-Pass Converter Boundaries

Good candidates

  • Process flows
  • Decision trees
  • Routing diagrams
  • API request paths
  • Data pipeline diagrams
  • Simple system dependency maps

Skip initially

  • Subgraphs and swimlanes
  • Sequence diagrams
  • Class diagrams
  • Custom Mermaid styling
  • Markdown-heavy labels
  • Complex curved or multi-segment routing
